## 🛠️ About

All circuits were assembled by hand on breadboard or PCB.
Videos demonstrate real hardware in operation — no simulations.

**Topics covered:**
- Arduino microcontroller programming (C/C++)
- Analog circuit design (NE555, oscillators)
- Sensor integration (PIR, ultrasonic, light, water level)
- Boolean logic gate implementations
- LED sequencing and timing circuits
